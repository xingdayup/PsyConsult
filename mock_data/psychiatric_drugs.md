# 精神科常用药物手册

## SSRI 类抗抑郁药

### 舍曲林 (Sertraline)
- 类别: SSRI
- 适应症: 抑郁障碍、强迫症、惊恐障碍、社交焦虑障碍、创伤后应激障碍
- 剂量: 起始 50mg/d，根据反应 2-4 周后可增至 100-200mg/d，最大 200mg/d
- 常见副作用: 恶心、腹泻、失眠、性功能障碍（延迟射精、性欲减退）
- 禁忌: 禁止与 MAOIs（单胺氧化酶抑制剂）联用，需 14 天洗脱期
- 相互作用:
  - 与其他 SSRI 或 SNRI 联用: 增加 5-HT 综合征风险（禁忌）
  - 与 NSAIDs 联用: 增加消化道出血风险（谨慎）
  - 与华法林联用: 可能增强抗凝作用（谨慎）

### 帕罗西汀 (Paroxetine)
- 类别: SSRI
- 适应症: 抑郁障碍、广泛性焦虑障碍、社交焦虑障碍、惊恐障碍
- 剂量: 起始 20mg/d，最大 50mg/d
- 常见副作用: 嗜睡、体重增加、性功能障碍、撤药综合征（眩晕、感觉异常）
- 禁忌: 禁止与 MAOIs 联用
- 相互作用:
  - 与舍曲林等其他 SSRI 联用: 5-HT 综合征风险（禁忌）
  - 与三环类抗抑郁药联用: 抑制 CYP2D6 代谢，可能导致三环类药物血药浓度升高（谨慎）

### 氟西汀 (Fluoxetine)
- 类别: SSRI
- 适应症: 抑郁障碍、强迫症、神经性贪食
- 剂量: 起始 20mg/d，最大 80mg/d
- 常见副作用: 失眠、焦虑（用药初期）、食欲减退、性功能障碍
- 禁忌: 禁止与 MAOIs 联用（需 5 周洗脱期，因氟西汀半衰期长）

## SNRI 类抗抑郁药

### 文拉法辛 (Venlafaxine)
- 类别: SNRI
- 适应症: 抑郁障碍、广泛性焦虑障碍、社交焦虑障碍
- 剂量: 起始 75mg/d，最大 225mg/d
- 常见副作用: 恶心、失眠、血压升高（高剂量时）、撤药综合征
- 禁忌: 未控制的高血压
- 相互作用:
  - 与 MAOIs 联用: 禁忌
  - 与含血清素药物联用: 5-HT 综合征风险

### 度洛西汀 (Duloxetine)
- 类别: SNRI
- 适应症: 抑郁障碍、广泛性焦虑障碍、糖尿病神经病变疼痛
- 剂量: 起始 30-60mg/d，最大 120mg/d
- 常见副作用: 恶心、口干、失眠
- 禁忌: 肝功能不全

## 非典型抗精神病药

### 奥氮平 (Olanzapine)
- 类别: 非典型抗精神病药
- 适应症: 精神分裂症、双相躁狂发作、双相障碍维持治疗
- 剂量: 起始 5-10mg/d，最大 20mg/d
- 常见副作用: 体重增加（显著）、嗜睡、血糖升高、血脂异常
- 相互作用:
  - 与氟西汀联用: 奥氮平血药浓度升高
  - 与利培酮联用: 无禁忌但需注意 EPS 风险叠加

### 利培酮 (Risperidone)
- 类别: 非典型抗精神病药
- 适应症: 精神分裂症、双相躁狂发作
- 剂量: 起始 1-2mg/d，最大 6mg/d
- 常见副作用: 锥体外系反应(EPS)、高催乳素血症、体重增加
- 相互作用:
  - 与舍曲林联用: 利培酮血药浓度升高

## 心境稳定剂

### 锂盐 (Lithium)
- 类别: 心境稳定剂
- 适应症: 双相障碍躁狂发作、双相障碍维持治疗、难治性抑郁增效治疗
- 剂量: 起始 300-600mg/d，根据血锂浓度调整，治疗窗 0.6-1.2 mmol/L
- 常见副作用: 震颤、多尿、甲状腺功能减退、体重增加
- 禁忌: 严重肾功能不全、低钠血症
- 相互作用:
  - 与 NSAIDs（布洛芬等）联用: 血锂浓度升高（谨慎，需监测）
  - 与噻嗪类利尿剂联用: 血锂浓度显著升高（禁忌）
  - 与奥氮平联用: 协同增效，可用于难治性双相（谨慎，注意神经毒性）
  - 与舍曲林联用: 可用于难治性抑郁增效治疗（谨慎）

### 丙戊酸钠 (Valproate)
- 类别: 心境稳定剂/抗癫痫药
- 适应症: 双相躁狂发作、双相障碍维持治疗
- 剂量: 起始 500-750mg/d，根据血药浓度调整，治疗窗 50-100 μg/mL
- 常见副作用: 体重增加、脱发、震颤、肝功能异常、多囊卵巢综合征
- 禁忌: 肝功能严重受损、育龄女性（致畸风险）
- 相互作用:
  - 与拉莫三嗪联用: 丙戊酸钠升高拉莫三嗪血药浓度（谨慎，需减量）
  - 与奥氮平联用: 协同增效，可用于躁狂治疗（谨慎，注意镇静作用）
  - 与舍曲林联用: 无直接禁忌，但需监测肝功能

## 心境稳定剂 / 抗癫痫药

### 拉莫三嗪 (Lamotrigine)
- 类别: 抗癫痫药/心境稳定剂
- 适应症: 双相障碍抑郁发作、双相障碍维持治疗（预防抑郁复发）
- 剂量: 起始 25mg/d，缓慢滴定至 100-200mg/d（快速加量增加皮疹风险）
- 常见副作用: 皮疹（包括Stevens-Johnson综合征）、头痛、恶心
- 禁忌: 对拉莫三嗪过敏
- 相互作用:
  - 与丙戊酸钠联用: 拉莫三嗪血药浓度升高（谨慎，需减半剂量）
  - 与舍曲林联用: 无明显相互作用（安全）

## 非典型抗精神病药（续）

### 喹硫平 (Quetiapine)
- 类别: 非典型抗精神病药
- 适应症: 精神分裂症、双相躁狂发作、双相抑郁发作、难治性抑郁增效治疗
- 剂量: 精神分裂症 400-800mg/d；双相抑郁 300mg/d；抑郁增效 150-300mg/d
- 常见副作用: 嗜睡、体重增加、口干、头晕、体位性低血压
- 禁忌: 未纠正的低钾血症或低镁血症（QT间期延长风险）
- 相互作用:
  - 与舍曲林联用: 可用于难治性抑郁（谨慎，注意镇静作用）
  - 与奥氮平联用: 同为非典型抗精神病药，无明显禁忌但注意体重增加和代谢副作用叠加
  - 与锂盐联用: 可用于躁狂发作（谨慎，注意神经毒性）

### 阿立哌唑 (Aripiprazole)
- 类别: 非典型抗精神病药（多巴胺部分激动剂）
- 适应症: 精神分裂症、双相躁狂发作、难治性抑郁增效治疗
- 剂量: 精神分裂症 10-30mg/d；抑郁增效 2-15mg/d
- 常见副作用: 静坐不能、失眠、恶心、头痛；体重增加风险低
- 禁忌: 对阿立哌唑过敏
- 相互作用:
  - 与舍曲林联用: 可用于难治性抑郁的增效治疗（谨慎，舍曲林升高阿立哌唑血药浓度）
  - 与奥氮平联用: 同为抗精神病药，联合使用需评估必要性

### 氯氮平 (Clozapine)
- 类别: 非典型抗精神病药
- 适应症: 难治性精神分裂症（至少 2 种抗精神病药足量足疗程治疗无效）
- 剂量: 起始 12.5-25mg/d，缓慢滴定至 300-600mg/d，最大 900mg/d
- 常见副作用: 粒细胞缺乏症（需定期监测血常规，前 18 周每周一次）、嗜睡、流涎、体重增加（显著）、心肌炎风险、癫痫发作
- 禁忌: 粒细胞缺乏症史、未控制的癫痫、严重心脏疾病
- 相互作用:
  - 与锂盐联用: 增加神经毒性风险（谨慎）
  - 与舍曲林联用: 舍曲林升高氯氮平血药浓度（谨慎，需监测）
  - 与奥氮平联用: 同为抗精神病药，无临床联用指征

## SSRI 类抗抑郁药（续）

### 氟伏沙明 (Fluvoxamine)
- 类别: SSRI
- 适应症: 强迫症、抑郁障碍、社交焦虑障碍
- 剂量: 起始 50mg/d，最大 300mg/d（OCD常用较高剂量）
- 常见副作用: 恶心、嗜睡、乏力、口干、失眠
- 禁忌: 禁止与 MAOIs 联用；禁止与替扎尼定联用
- 相互作用:
  - 与氯氮平联用: 显著升高氯氮平血药浓度（禁忌）
  - 与阿立哌唑联用: 升高阿立哌唑血药浓度（谨慎）
  - 与其他 SSRI 联用: 5-HT 综合征风险（禁忌）

## 药物相互作用风险等级定义
- 🔴 **禁忌 (Contraindicated)**: 绝对禁止联用（如 SSRI + MAOI → 5-HT 综合征风险）
- 🟡 **谨慎 (Use with Caution)**: 可联用但需监测血药浓度或调整剂量
- 🟢 **安全 (Safe)**: 常规剂量下无已知严重相互作用
