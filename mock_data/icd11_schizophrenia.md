# 精神分裂症 ICD-11 诊断标准

## 6A20 精神分裂症

### 核心标准（症状持续 ≥1 月，至少符合 2 条）
1. 持续性幻觉（幻听为主，如议论性、命令性幻听）
2. 持续性妄想（关系妄想、被害妄想、被控制感等）
3. 思维紊乱（言语离题、破裂、不连贯）
4. 被动体验（思维被插入、被广播、被抽出、被控制）
5. 阴性症状（情感淡漠、意志减退、社交退缩、言语贫乏）
6. 行为紊乱（紧张症、不适当的情绪反应）

### 排除标准
- 并非由物质使用或躯体疾病引起
- 症状不是双相障碍/抑郁障碍伴精神病性症状的表现

### 临床分型
- 首次发作: 6A20.0
- 多次发作: 6A20.1
- 持续型: 6A20.2

### 鉴别要点
- 与双相躁狂的鉴别：精分的情感症状不占主导，病程中大部分时间为精神病性症状
- 与药物所致精神障碍的鉴别：时间关联性和停药后缓解
