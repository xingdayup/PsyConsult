### 治疗指南（1 份）

`china_guidelines.md` 摘要了中国两大指南：

- **抑郁障碍防治指南（第二版）**：急性期→巩固期→维持期的治疗路径，SSRI/SNRI 一线推荐，难治性抑郁的增效策略
- **精神分裂症防治指南（第二版）**：非典型抗精神病药首选，维持期 ≥2 年，长效针剂提高依从性
- **双相障碍治疗原则**：心境稳定剂（锂盐/丙戊酸钠）+ 抗精神病药，抑郁期谨慎使用抗抑郁药
- **焦虑障碍治疗原则**：SSRI/SNRI 一线，苯二氮卓短期辅助，CBT 心理治疗
- **强迫症治疗原则**：SSRI（高剂量）+ ERP 暴露反应预防，难治性加非典型抗精神病药增效

## 数据流向

```
mock_data/*.md
    │
    ├──→ build_kg.py
    │     LLM 逐块提取实体+关系 → JSON → Neo4j（知识图谱）
    │
    └──→ milvus_rag.py
          分块(chunk_size=500) → 嵌入 → Milvus（向量 RAG）

symptom_synonyms.json
    │
    └──→ synonym_tool.py
          症状 Agent 加载到内存，做口语→术语映射
```

## 如何扩展

需要新增一种疾病（如 PTSD）：

1. 写 `icd11_ptsd.md`，按 ICD-11 标准格式
2. `python test/build_kg.py ../mock_data/icd11_ptsd.md` → 导入图谱
3. `python test/milvus_rag.py` → 重新摄入向量库
4. 在 `symptom_synonyms.json` 里加 PTSD 相关口语映射

不需要改任何代码。
